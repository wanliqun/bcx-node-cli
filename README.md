# Usage
<!-- usage -->
```sh-session
$ npm install -g bcxcli
$ bcxcli COMMAND
running command...
$ bcxcli (-v|--version|version)
bcxcli/0.0.1 linux-x64 node-v10.16.2
$ bcxcli --help [COMMAND]
USAGE
  $ bcxcli COMMAND
...
```
